## 特性

- 🎨 **无限画布布局** - 自由摆放卡片，无限扩展，支持缩放和平移
- 📐 **工作台布局** - 分区窗口组合，适合专注单卡片编辑
- 📁 **文件管理器** - 管理卡片和箱子文件
- ✏️ **编辑面板** - 通过插件编辑基础卡片
- 📦 **卡箱库** - 拖放创建 26 种基础卡片和 8 种箱子布局
- ↩️ **撤销重做** - 完整的操作历史支持
- 🎯 **拖放系统** - 丰富的拖放交互
- 🔌 **插件系统** - 可扩展的编辑组件

## 技术栈

- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **构建工具**: Vite
- **测试**: Vitest + Vue Test Utils
- **桌面封装**: Electron

## 开发规范

本项目遵循薯片生态的开发规范，包括：

- **中心路由**: 所有通信通过 Chips Core
- **多语言**: 所有文本使用 i18n
- **配置化**: 所有配置通过配置系统
- **主题支持**: 使用 CSS 变量，不硬编码样式
- **测试覆盖**: 核心模块 >= 80%

详见 [开发规范总则](../生态共用/08-开发规范总则.md)
