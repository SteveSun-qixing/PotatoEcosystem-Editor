## 界面介绍

### 无限画布布局

无限画布布局是默认的编辑模式，提供自由的卡片摆放空间。

```
┌─────────────────────────────────────────────────────────────┐
│ ┌─────────────────────────────────────────────────────────┐ │
│ │                        桌面层                           │ │
│ │                                                         │ │
│ │    ┌──────────┐    ┌──────────┐    ┌──────────┐       │ │
│ │    │  卡片窗口 │    │  卡片窗口 │    │ 工具窗口  │       │ │
│ │    │          │    │          │    │ (文件管理) │       │ │
│ │    │          │    │          │    │          │       │ │
│ │    └──────────┘    └──────────┘    └──────────┘       │ │
│ │                                                         │ │
│ │                                                         │ │
│ │                                    ┌──────────┐        │ │
│ │                                    │  卡箱库   │        │ │
│ │                                    └──────────┘        │ │
│ │                                                         │ │
│ └─────────────────────────────────────────────────────────┘ │
│ ┌──────────────────────────────────────┐  ┌─────────────┐   │
│ │          程序坞 (Dock)                │  │ 缩放: 100%  │   │
│ └──────────────────────────────────────┘  └─────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

**主要组成部分**:

| 组件 | 描述 |
|------|------|
| **桌面层** | 可无限扩展的画布背景，支持平移和缩放 |
| **卡片窗口** | 显示和编辑单个卡片的窗口 |
| **工具窗口** | 文件管理器、编辑面板、卡箱库等工具 |
| **程序坞** | 显示工具图标，快速打开工具窗口 |
| **缩放控制器** | 显示当前缩放级别，支持快速调整 |

```
│ │   └📁 项目 │  │                     ││   标签: 工作       │ │
│ │           │  │                     ││                   │ │

│ │           │  │                     ││   1. Markdown     │ │

└─────────────────────────────────────────────────────────────┘
```

**主要组成部分**:

| 区域 | 描述 |
|------|------|
| **左侧面板** | 文件管理器，浏览和管理卡片文件 |
| **主编辑区** | 显示当前编辑的卡片内容 |
| **右侧面板** | 属性面板，编辑卡片属性和基础卡片 |
| **底部面板** | 历史记录面板，查看操作历史 |

卡片窗口是编辑卡片的主要界面。

```

│ ┌──────────────────────────────────────────────────────┐ │
│ │  📄 我的笔记                    ━  ⬜  ✕             │ │  ← 标题栏
│ └──────────────────────────────────────────────────────┘ │
│ ┌──────────────────────────────────────────────────────┐ │
│ │                                                      │ │
│ │  # 这是标题                                          │ │
│ │                                                      │ │  ← 卡片内容
│ │  这是正文内容...                                     │ │
│ │                                                      │ │
│ │  ┌────────────────────────────────────────────────┐ │ │
│ │  │             [图片内容]                          │ │ │
│ │  └────────────────────────────────────────────────┘ │ │
│ │                                                      │ │
│ └──────────────────────────────────────────────────────┘ │
```

**窗口标题栏功能**:

| 按钮 | 功能 |
|------|------|
| **卡片名称** | 双击可编辑卡片名称 |
| **━** | 最小化窗口到程序坞 |
| **⬜** | 折叠/展开窗口 |
| **✕** | 关闭窗口 |

### 工具窗口

#### 文件管理器

管理卡片和箱子文件。

- 树形结构展示文件
- 支持搜索文件
- 右键菜单操作（新建、重命名、删除）
- 拖放文件到画布

#### 编辑面板

编辑选中的基础卡片。

- 显示基础卡片的编辑组件
- 实时预览编辑效果
- 支持插件扩展

#### 卡箱库

浏览和创建基础卡片类型。

**基础卡片类型（26种）**:
- 文本类: Markdown、富文本、代码
- 媒体类: 图片、视频、音频
- 数据类: 表格、图表、公式
- 交互类: 按钮、表单、链接
- 更多...

**箱子布局类型（8种）**:
- 列表布局
- 网格布局
- 瀑布流布局
- 卡片堆叠布局
- 更多...

#### 程序坞

快速访问工具窗口。

- 显示工具图标
- 点击图标打开/聚焦工具窗口
- 显示最小化的工具窗口

## 基本操作

### 画布操作

#### 平移画布

| 方式 | 操作 |
|------|------|
| **鼠标拖拽** | 按住鼠标左键拖拽空白区域 |
| **触摸板** | 双指拖动 |
| **方向键** | 按住 Shift + 方向键 |

#### 缩放画布

| 方式 | 操作 |
|------|------|
| **鼠标滚轮** | Ctrl/Cmd + 滚轮 |
| **触摸板** | 双指捏合 |
| **缩放控制器** | 点击 +/- 按钮或拖动滑块 |
| **快捷键** | Ctrl/Cmd + 0（重置为100%）|

**缩放范围**: 10% - 500%

---

### 窗口操作

#### 移动窗口

- 拖拽窗口标题栏移动窗口位置

#### 调整大小

- 拖拽窗口边缘或角落调整大小

#### 窗口层级

- 点击窗口将其提升到最前
- 使用窗口菜单选择"置顶"

#### 窗口状态

| 操作 | 描述 |
|------|------|
| **最小化** | 窗口缩小到程序坞 |
| **折叠** | 只显示窗口标题栏 |
| **关闭** | 关闭窗口 |

---

### 卡片操作

#### 创建卡片

**方法 1: 从卡箱库拖放**

1. 打开卡箱库（程序坞中点击）
2. 选择基础卡片类型
3. 拖放到画布空白区域
4. 自动创建包含该基础卡片的新卡片

**方法 2: 使用菜单**

1. 右键点击画布空白区域
2. 选择"新建卡片"
3. 输入卡片名称

**方法 3: 快捷键**

- `Ctrl/Cmd + N` 创建新卡片

#### 打开卡片

- 在文件管理器中双击卡片文件
- 拖放卡片文件到画布

#### 保存卡片

卡片会自动保存。也可以手动保存：
- `Ctrl/Cmd + S` 保存当前卡片
- `Ctrl/Cmd + Shift + S` 保存所有卡片

#### 关闭卡片

- 点击窗口的关闭按钮
- 如果有未保存的更改，会提示保存

#### 删除卡片

1. 在文件管理器中右键点击卡片
2. 选择"删除"
3. 确认删除

---

### 基础卡片操作

#### 添加基础卡片

**方法 1: 从卡箱库拖放**

1. 从卡箱库拖动基础卡片类型
2. 放置到已有卡片窗口中
3. 基础卡片添加到卡片末尾

**方法 2: 使用工具栏**

1. 在卡片窗口中点击"+"按钮
2. 选择基础卡片类型

#### 编辑基础卡片

1. 点击卡片中的基础卡片
2. 编辑面板显示该基础卡片的编辑组件
3. 编辑内容实时保存

#### 排序基础卡片

- 拖动基础卡片到新位置
- 释放时自动更新顺序

#### 删除基础卡片

1. 选中基础卡片
2. 按 Delete 键删除
3. 或右键选择"删除"

---

### 文件操作

#### 导入文件

1. 从系统文件管理器拖动文件到画布
2. 根据文件类型自动创建对应卡片：
   - 图片 → 图片卡片
   - 视频 → 视频卡片
   - Markdown → Markdown 卡片
   - .chip 文件 → 打开卡片

#### 导出卡片

1. 右键点击卡片窗口
2. 选择"导出"
3. 选择导出格式（PDF、Markdown、HTML 等）
4. 选择保存位置

## 高级功能

### 撤销和重做

编辑器支持完整的撤销/重做功能。

| 操作 | 快捷键 |
|------|--------|
| 撤销 | `Ctrl/Cmd + Z` |
| 重做 | `Ctrl/Cmd + Shift + Z` 或 `Ctrl/Cmd + Y` |

**查看历史记录**:

1. 打开历史记录面板（程序坞中点击）
2. 查看操作列表
3. 点击任意历史节点跳转

---

### 布局切换

在无限画布和工作台布局之间切换。

1. 点击工具栏的布局切换按钮
2. 或使用快捷键 `Ctrl/Cmd + L`

切换时会自动保存当前布局状态，再次切换时恢复。

---

### 窗口布局

#### 平铺窗口

自动排列所有窗口为网格布局。

1. 菜单 → 窗口 → 平铺窗口
2. 或使用快捷键 `Ctrl/Cmd + Shift + T`

#### 层叠窗口

自动排列所有窗口为层叠布局。

1. 菜单 → 窗口 → 层叠窗口
2. 或使用快捷键 `Ctrl/Cmd + Shift + C`

---

### 卡片嵌套

将一个卡片拖放到另一个卡片中，创建嵌套结构。

1. 拖动卡片窗口
2. 悬停在目标卡片上
3. 出现嵌套提示时释放
4. 卡片成为目标卡片的子卡片

---

### 搜索

全局搜索卡片和内容。

1. 按 `Ctrl/Cmd + P` 打开搜索框
2. 输入搜索关键词
3. 选择搜索结果打开对应卡片

---

### 多选操作

#### 框选

1. 在画布空白区域按住鼠标
2. 拖动绘制选择框
3. 释放时选中框内所有窗口

#### 添加选择

- 按住 `Ctrl/Cmd` 点击窗口添加到选择
- 再次点击取消选择

#### 批量操作

选中多个窗口后可进行批量操作：
- 移动（拖动任一选中窗口）
- 关闭（按 Delete）
- 平铺

## 常见问题

### 问：如何恢复误删的卡片？

**答**：删除的卡片会移动到回收站，可以通过以下步骤恢复：

1. 打开文件管理器
2. 点击"回收站"
3. 找到要恢复的卡片
4. 右键选择"恢复"

注意：回收站中的文件会在 30 天后自动永久删除。

---

### 问：编辑器卡顿怎么办？

**答**：尝试以下方法：

1. **减少打开的窗口数量**
   - 关闭不需要的窗口
   - 使用工作台布局专注单个卡片

2. **缩小缩放级别**
   - 在较小的缩放级别下，编辑器会简化渲染

3. **检查卡片内容**
   - 大型图片或视频可能影响性能
   - 考虑压缩媒体文件

4. **重启编辑器**
   - 清除缓存后重新打开

---

### 问：卡片保存失败怎么办？

**答**：检查以下几点：

1. **磁盘空间**
   - 确保有足够的存储空间

2. **文件权限**
   - 确保有写入权限

3. **网络连接**
   - 如果使用云同步，检查网络连接

4. **手动保存**
   - 尝试使用 `Ctrl/Cmd + S` 手动保存
   - 查看错误提示信息

---

### 问：如何导入其他格式的文件？

**答**：支持的导入格式：

| 格式 | 处理方式 |
|------|----------|
| `.chip` | 直接打开卡片 |
| `.box` | 打开箱子 |
| `.md` / `.markdown` | 创建 Markdown 卡片 |
| `.jpg` / `.png` / `.gif` | 创建图片卡片 |
| `.mp4` / `.webm` | 创建视频卡片 |
| `.mp3` / `.wav` | 创建音频卡片 |
| `.pdf` | 创建 PDF 卡片 |

直接将文件拖放到画布即可导入。

---

### 问：如何同步卡片到其他设备？

**答**：

1. **使用云存储**
   - 将工作目录设置在云存储文件夹中（如 Dropbox、iCloud、OneDrive）
   - 卡片会自动同步到云端

2. **导出和导入**
   - 导出卡片为 `.chip` 文件
   - 在其他设备上导入

---

### 问：插件如何安装？

**答**：

1. 打开设置（`Ctrl/Cmd + ,`）
2. 选择"插件"选项卡
3. 浏览插件市场
4. 点击"安装"按钮
5. 重启编辑器生效

---

### 问：如何自定义快捷键？

**答**：

1. 打开设置（`Ctrl/Cmd + ,`）
2. 选择"快捷键"选项卡
3. 搜索要修改的命令
4. 点击快捷键输入框
5. 按下新的快捷键组合
6. 保存设置
